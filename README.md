# My Knowledge App

A personal knowledge vault web app that connects to my private GitHub repository to store, browse, and search all my ideas, learnings, and inspirations.

## Features

- Browse all notes with category filters (9 categories)
- Instant full-text search across all content
- Quick-add form to save new ideas from any device
- Clean, mobile-friendly design
- Powered entirely by the GitHub API — no backend needed

## Setup

1. Open `index.html` in any browser (or deploy to GitHub Pages)
2. Enter your [GitHub Personal Access Token](https://github.com/settings/tokens/new?scopes=repo&description=My+Ideas+Vault) with `repo` scope
3. Your vault loads instantly — token is saved in your browser

## Tech

Vanilla HTML, CSS, and JavaScript. No frameworks, no build step, no server needed.
